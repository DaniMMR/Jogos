# 🛡️ Shield Sprint (ou o nome que você escolheu)

Um jogo de corrida infinita baseado em física real, desenvolvido com HTML5 Canvas e JavaScript puro.

## 🚀 Como Jogar
Acesse o jogo aqui: [Link do seu GitHub Pages]

### Comandos:
- **WASD / Setas**: Movimentação e Pulo.
- **Espaço**: Pulo Duplo.
- **Shift**: Correr (Sprint).

## 🛠️ Tecnologias
- HTML5 Canvas
- CSS3 (UI e Overlays)
- JavaScript Vanilla (Motor de Física, Partículas e Colisões)

## 🌟 Destaques
- Sistema de inércia e fricção realista.
- Escudo automático ativado após 200m de distância.
- Dificuldade progressiva conforme o cenário acelera.
